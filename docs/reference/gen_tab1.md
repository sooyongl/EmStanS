# ESS analysis for overall GCA

ESS analysis for overall GCA

## Usage

``` r
gen_tab1(tab0, information)
```

## Arguments

- tab0:

  a list of data coming from
  [`gen_tab0`](https://sooyongl.github.io/EmStanS/reference/gen_tab0.md)

- information:

  a list of data coming from
  [`get_data_info`](https://sooyongl.github.io/EmStanS/reference/get_data_info.md)

## Value

a list of data for each GCA aggregating all the panelists and the tables
about ESS results.
