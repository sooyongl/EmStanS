# Generate item distribution plots

Generate item distribution plots

## Usage

``` r
plot_dist_items(information, x_name = "", y_name = "", font_size = 16)
```

## Arguments

- information:

  a list of data generated by
  [`get_data_info`](https://sooyongl.github.io/EmStanS/reference/get_data_info.md)

- x_name:

  a character for y-axis title

- font_size:

  an integer for font size of plot

## Value

a list of item distribution plots for each GCA
