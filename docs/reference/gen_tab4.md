# Item review table

Item review table

## Usage

``` r
gen_tab4(tab1, tab2, information)
```

## Arguments

- tab1:

  a list of data coming from
  [`gen_tab1`](https://sooyongl.github.io/EmStanS/reference/gen_tab1.md)

- tab2:

  a list of data coming from
  [`gen_tab2`](https://sooyongl.github.io/EmStanS/reference/gen_tab2.md)

- information:

  a list of data coming from
  [`get_data_info`](https://sooyongl.github.io/EmStanS/reference/get_data_info.md)

## Value

a list containing item review tables
