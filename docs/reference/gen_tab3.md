# Cut score summary

Cut score summary

## Usage

``` r
gen_tab3(tab1, information)
```

## Arguments

- tab1:

  a list of data coming from
  [`gen_tab1`](https://sooyongl.github.io/EmStanS/reference/gen_tab1.md)

- information:

  a list of data coming from
  [`get_data_info`](https://sooyongl.github.io/EmStanS/reference/get_data_info.md)

## Value

a list containing cut score summaries for each domain
