# report_itemtable1

The function provides the table for the number of items in the rating
data when the data is imported.

## Usage

``` r
report_itemtable1(imprt_data)
```

## Arguments

- imprt_data:

  a list of data from
  [`read_data`](https://sooyongl.github.io/EmStanS/reference/read_data.md)

## Value

kable output
