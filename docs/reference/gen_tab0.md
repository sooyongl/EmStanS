# ESS analysis for individual panelists

ESS analysis for individual panelists

## Usage

``` r
gen_tab0(information)
```

## Arguments

- information:

  a list of data generated by
  [`get_data_info`](https://sooyongl.github.io/EmStanS/reference/get_data_info.md)

## Value

a list of data for each panelist about ESS results.
