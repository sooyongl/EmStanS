# Check input data for validation

Check input data for validation

## Usage

``` r
validateData(imprt_data)
```

## Arguments

- imprt_data:

  a list containing imported data such as setup, panelists, rating, item
  meta, and examinee data, which comes from
  [`read_data`](https://sooyongl.github.io/EmStanS/reference/read_data.md).

## Value

a list containing messages related to validation.
